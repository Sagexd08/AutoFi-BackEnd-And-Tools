# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-15

### Added
- Initial release of Celo AI SDK
- Multi-chain support (Ethereum, Polygon, BSC, Arbitrum, Optimism, Celo, Base, Avalanche)
- AI agent system with Gemini AI integration and intelligent agent orchestration
- Dynamic smart contract deployment and management with contract factory and compilation support
- Load balancing and proxy server capabilities
- Postman protocol integration for API testing
- Comprehensive error handling and validation
- Health monitoring and metrics
- Circuit breaker patterns for reliability
- Event-driven architecture
- AutoFi SDK wrapper for branded access
- TypeScript support with full type definitions
